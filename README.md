# JavaScript Questions

### 1. What is the difference between var, let, and const?
var is function-scoped and can be re-declared, which often causes bugs. let is block-scoped and allows value updates but not re-declaration. const is also block-scoped but remains fixed and cannot be reassigned.
### 2. What is the spread operator (...)?
The spread operator uses three dots to unpack or spread elements of an array or object. It is the most efficient way to copy data or merge multiple arrays and objects into one.
### 3. What is the difference between map(), filter(), and forEach()?
forEach is a simple loop that performs an action but returns nothing. map transforms every element and returns a new array. filter selects specific elements based on a condition and returns them in a new array.
### 4. What is an arrow function?
An arrow function is a modern, shorthand way to write functions using the => syntax. It makes the code more readable by removing the need for the function keyword and often the return statement.
### 5. What are template literals?
Template literals are strings wrapped in backticks (`) instead of quotes. They allow you to easily inject variables directly into a string using the ${} syntax and support multi-line text.
