# 🛠️ GitHub Issues Tracker

A sleek, minimalist dashboard designed to manage and track repository issues with ease, built with a focus on clean UI and efficient task monitoring.

🔗 **Live Demo:** https://t4wchif.github.io/GitHub-Issues-Tracker/

### ✨ Key Features
- **Secure Login:** Dedicated authentication layer for protected access.
- **Issue Dashboard:** Overview of all active and resolved tasks.
- **Dynamic Filtering:** Sort issues by status (Open/Closed/All).
- **Responsive Design:** Fully optimized for mobile and desktop browsers.
- **Minimalist UI:** Distraction-free interface for better productivity.

---

### 🛠️ Tech Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Tailwind CSS)
- **Logic:** Vanilla JavaScript (ES6+)
- **Hosting:** GitHub Pages
